## About the Project

**WhatToWear (W2W)** is a fashion-tech mobile app concept that helps users decide what to wear every day. It combines wardrobe management, AI-powered outfit suggestions, real-time weather integration, and a social community — all in one place.

### Core Features
- **Daily outfit suggestions** based on real-time weather data
- **Outfit builder** — generate a full look starting from a single garment
- **Digital wardrobe** — upload and organize your clothes by category, color, and style
- **Community (Explore)** — share outfits, browse others' looks, follow users
- **Context-aware planning** — specify occasion, location, date, and time for tailored suggestions

---

## Design & Evaluation Process

The project followed a full HCI design cycle:

1. **Requirement Analysis** — user questionnaire (45 participants), personas, scenarios, competitor analysis
2. **Task Analysis** — HTA and STN diagrams for 3 key user flows
3. **Wireframes & Mockups** — low and high fidelity design in Figma
4. **Expert-Based Evaluation** — Heuristic Evaluation + Cognitive Walkthrough (by course professors)
5. **User-Based Evaluation** — Think Aloud session + Controlled Experiment (icons-only vs. icons+labels, analyzed with one-way ANOVA)
